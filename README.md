# AGM Search
## 概要
アナログゲームミュージアムの所蔵資料のメタデータ、デジタイズしたコンテンツ及びそれらの公開に関するコードなどを試験的に公開しています。

## ライセンス
CC0: バックアップ・エンドポイントで公開するデータセット
